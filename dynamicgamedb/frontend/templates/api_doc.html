Resource Description Parameters Respsonses Authentication
GET /api/games
Returns all games in the database
JSON

A list of JSON information about all games in the database

Example Values:
{"games":
[{"title": "Zelda: A Link to the Past",
"platform" : "SNES"},
{"title": "Diablo 2",
"platform" : "PC"}]}

 None
GET /api/game/:id/
Returns a game specified with the id parameter
id required

Specific identifier for the game selected

Example Values: 123
JSON

Json message with information about specified game

Example Values:
{"title": "Zelda: A Link to the Past",
"platform" : "SNES"}

 None
POST /api/game/add
Add a game with a specified title to a specified platform
title required

The title specified in the Add game form input field

Example Values: "Zelda: A Link to the Past"

platform required

The platform that was specified to add the game on

Example Values: "SNES"
200, OK

If game was successfully added

Error

Game allready exists for specified platform

User required

Needs to be logged in as a user in order to edit games

POST /api/game/:id/edit
Edit/Add additional information about the game specified with the id
Parameters
id required

Id for the specific game that should be edited

Example Values: 123

Responses
200, OK

If game was successfully edited

Error

No such game_id exists

Authentication
User required

Needs to be logged in as a user in order to edit games

POST /api/game/:id/connection
Creates a connect between the id specified and the game choose to make connection towards on the webpage
